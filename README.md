## Tezpur University Subject Lists

**JSON** for Tezpur University subjects  using 2023 Prospectus.
It demonstrates how to load, decode, and display structured subject data from a JSON file using pure Dart and python.


# Features
-  Reads data from a local JSON file (`subjects.json`)
-  Parses subject details (Course Code & Course Name)
-  Converts JSON into any PL objects for easy handling
-  Displays formatted output in the console
-  Example of searching/filtering through subjects
-  Lightweight and easy to integrate into other Dart or backend systems
